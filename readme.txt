/////////////////////////////////////////////////////////////////

Create Sankey Diagram to Visualize Flow of Refugees Over Different Years

/////////////////////////////////////////////////////////////////////


Assignment Description:
Create a Sankey Chart to visualize country of origin, asylum and number of refugees over the years 2005-2014. 
The chart visualizes the flow of refugees from one country to another over the years. 
A slider is used to navigate over the years. Countries are sorted in alphabetical order. Tooltips are used to show more info.


Two sample screenshots are shown.


Setup:
May need to setup your own HTTP server to properly run the D3 visualizations. The easiest way is using SimpleHTTPServer in Python. 
viz.html  is the html to render the visualization.
Input Data file: poc.json
lib folder includes the libraries used.
